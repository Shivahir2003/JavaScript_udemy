Loops in javaScript

==> Loops can execute a block of code a number of times.


  1. ==>JavaScript supports different kinds of loops:
          
    • While Loop
    • For Loop
    • Do While Loop


        While Loop

        var a=1;
        while(a <= 10)
        {
            document.write(a + ")Hello< br>");
            a++;
        }
            
        

    Do While Loop

        var a = 1;
        do {
            document.write(a + ")Hello< br>");
            a++;
        } while (a <= 10);


    For Loop

        for(var a=1; a<=10; a++)
        {
            document.write(a + ")Hello< br>");
        }


    nested Loop

        for (var a = 1; a <= 5; a++) 
        {
            for(var b = 1; b <= a; b++) 
            {
                document.write("* ");
            }
            document.write("< br>");
    }
